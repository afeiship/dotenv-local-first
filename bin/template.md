# dotenv-local-first
> Cli templates for javascript.

## template
```sh
# clone the template
git clone git@github.com:aric-tpls/dotenv-local-first.git
mvg

# or use tiged
tiged git@github.com:aric-tpls/dotenv-local-first.git --mode=git --disable-cache

# replace the template
rifc
```
